Personal Project — n8n • Supabase • AI • Python • TypeScript

>job — AI HR Assistant Bot
Personal Project
Situation

Modern job searching is often a numbers game where manually comparing profiles against hundreds of vacancies leads to fatigue and missed opportunities.

Task

The goal was to build an automated assistant that could objectively evaluate job compatibility, allowing for a more strategic and data-driven approach to career development.

Action

Architected an end-to-end automation pipeline using n8n. I configured Supabase for secure data persistence, developed custom Python scripts for advanced scoring logic, and integrated semantic analysis chains to match candidate strengths with vacancy requirements.

Result

Successfully deployed a functional HR assistant that generates structured compatibility reports. The tool has significantly optimized the application process, ensuring focus is directed toward the highest-probability opportunities.
